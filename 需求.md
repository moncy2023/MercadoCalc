# 美客多店铺利润核算系统 - 需求文档

## 一、项目概述
美客多(Mercado Libre)店铺利润核算工具，支持根据采购成本、运费、抽佣等因素，核算产品利润或反推平台售价。

## 二、技术要求
1. 语言要求：Python 3.10+
2. 每个源文件头需要说明 Python 语言版本
3. 配置文件采用 JSON 格式，方便修改和扩展

## 三、运行模式
1. **CLI 文本模式**：`python main.py cli`，从 config.json 读取参数并输出结果
2. **Web 模式**：`python main.py web`，启动 FastAPI 服务，浏览器访问 `http://127.0.0.1:9001`

## 四、核心功能

### 4.1 利润核算（两种方式）
1. **反推售价模式**：确定目标利润百分比，反推平台售价（USD）
   - 公式：`售价 = 固定成本 / (1 - 抽佣比例 - 退货率 - 目标利润率)`
2. **计算利润模式**：确定平台售价，计算目标利润率
   - 公式：`利润 = 售价 - (固定成本 + 售价 × 抽佣 + 售价 × 退货率)`

### 4.2 成本构成
- **采购成本**（含国内物流，单位：¥）
- **抽佣比例**（默认 15%）
- **退货率**（默认 2%，算入成本分摊）
- **平台重量/体积附加费**（当店铺填写的产品重量或体积不对时，平台收取的附加费）
- **运费**（详见 4.3）

### 4.3 运费计算
1. 每个国家一个独立的运费计算方法数据表（数据库中表名格式：`shipping_{国家名}`）
2. 运费标准从美客多官方 PDF 文件中提取，一次性写入数据库，与国家关联
3. 运费支持两套标准：**高于免邮阈值** 和 **低于免邮阈值**（如巴西 79 BRL）
4. 商品重量单位为 **克(g)**，系统自动转换为 kg 匹配数据库
5. 数据库中运费单位为 **USD**，系统通过页面汇率设置自动换算为 **RMB**
6. **最终运费公式**：`运费 = 重量 × 克重收费 × 汇率 - 平台运费折减`
7. **平台运费折减**：必须有独立函数 `calculate_platform_shipping_discount()` 计算，默认为 0，可设置
8. **【总运费】实时计算**：当国家、重量、汇率、阈值、折减等任一变量改动时，自动实时更新

### 4.4 货币与汇率
- 页面提供 **汇率换算** 输入框（1 USD = ? RMB），默认 7.2
- 核算结果中的 **平台售价** 显示为 **USD**，后附人民币参考值
- 其余成本项（运费、抽佣、利润等）显示为 **¥（人民币）**

### 4.5 国家元数据
- 每个国家存储免邮阈值信息（本币名称、本币金额、USD 等值）
- 页面中 "是否高于免邮阈值" 勾选框旁边，实时显示当前国家的阈值标签（如 `🏷️ BRL 79 ≈ USD 15`）
- 切换国家时阈值标签自动更新

## 五、Web 界面

### 5.1 利润核算页面（首页 `/`）
输入项：
- 采购成本（¥）
- 抽佣比例
- 退货率
- 国家/地区（下拉列表，从数据库动态加载，默认巴西）
- 汇率换算（1 USD = ? RMB）
- 是否高于免邮阈值（勾选框 + 动态阈值标签）
- 商品重量（g）
- 平台重量/体积附加费（¥）
- 平台运费折减（¥）
- 【总运费】自动计算（只读，实时更新，附计算明细）
- 目标利润百分比 / 平台售价（根据模式切换）

输出项：
- 平台售价（USD，附 ¥ 参考值）
- 【总运费】（¥，含折减明细）
- 固定成本（¥）
- 平台抽佣（¥）
- 退货成本（¥）
- 利润（¥）
- 利润率（%）

### 5.2 运费标准查看页面（`/shipping`）
- 选择国家，查询该国运费阶梯表
- 显示免邮阈值标签（本币 ≈ USD）
- 表格展示：序号、重量下限、重量上限、低于阈值运费、高于阈值运费

### 5.3 页面互通
- 利润核算页面提供 "📦 查看各国运费标准 →" 链接
- 运费标准页面提供 "← 返回利润核算" 链接

## 六、数据导入工具

### 6.1 PDF 运费导入（`import_pdf.py`）
```bash
python import_pdf.py --country Brazil --pdf brazil_shipping.pdf
```
- 从美客多官方 PDF 文件提取运费阶梯数据
- 自动清空该国旧数据，写入新数据
- 支持纯文本型 PDF 的自动正则解析

### 6.2 手动录入示例（`pdf_extractor.py`）
- 对于纯图片型 PDF（无法自动提取文本），已手动转录巴西完整运费数据
- 包含 25 条重量阶梯记录（0-15kg+），含高于/低于阈值两套费率

## 七、配置文件说明（config.json）
```json
{
    "procurement_cost": 50.0,           // 采购成本（¥）
    "manual_final_shipping_fee": 0.0,   // 手动运费覆盖（0=自动计算）
    "exchange_rate_usd_to_rmb": 7.2,    // 汇率
    "is_above_threshold": true,         // 是否高于免邮阈值
    "platform_shipping_discount": 0.0,  // 平台运费折减（¥）
    "return_rate": 0.02,                // 退货率
    "commission_rate": 0.15,            // 抽佣比例
    "additional_fee_for_wrong_weight": 0.0,  // 附加费（¥）
    "calculation_mode": 1,              // 1=反推售价, 2=计算利润
    "target_profit_percentage": 0.20,   // 目标利润率
    "platform_selling_price": 100.0,    // 平台售价（模式2用）
    "country_for_shipping": "Brazil",   // 默认国家
    "weight_g": 1000.0                  // 商品重量（克）
}
```
