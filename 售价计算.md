# 售价计算公式说明

## 一、核心公式

```
平台售价 (USD) = Net Proceeds + 平台佣金 + 物流费
```

其中：
- **平台佣金** = 佣金率 × 平台售价
- **物流费** = 从数据库按重量查表（USD）

### Net Proceeds 构成

```
Net Proceeds = 固定成本(USD) + 折损 + 目标净利润
```

| 项目 | 说明 | 单位 |
|------|------|------|
| 固定成本 | 采购成本 + 打包费（转换为 USD） | USD |
| 折损 | 折损率 × 平台售价 | USD |
| 目标净利润 | 目标利润率 × 平台售价 | USD |

### 比例费用（按售价百分比计算）

| 项目 | config 字段 | 默认值 |
|------|------------|-------|
| 平台佣金 | `commission_rate` | 20%（0.20） |
| 折损/退货率 | `loss_rate` | 2%（0.02） |
| 目标利润率 | `target_profit_percentage` | 18%（0.18） |

### 数学推导

```
P = (costs_rmb / rate) + loss_rate × P + profit_rate × P + commission_rate × P + shipping_usd
P × (1 - commission_rate - loss_rate - profit_rate) = (costs_rmb / rate) + shipping_usd
P = ((costs_rmb / rate) + shipping_usd) / (1 - commission_rate - loss_rate - profit_rate)
```

### 计算示例

> 采购成本 33¥，打包费 2¥，运费 USD 18.5（查表），佣金 20%，折损 2%，目标利润 18%，汇率 6.84

```
固定成本 = 33 + 2 = 35¥ → $5.12 USD
分母 = 1 - 0.20 - 0.02 - 0.18 = 0.60
平台售价 = (5.12 + 18.5) / 0.60 = $39.37 USD (≈ ¥269.27)

验证：
  Net Proceeds = $5.12 + $39.37×0.02 + $39.37×0.18 = $5.12 + $0.79 + $7.09 = $13.00
  平台佣金 = $39.37 × 0.20 = $7.87
  物流费 = $18.50
  合计 = $13.00 + $7.87 + $18.50 = $39.37 ✅
```

## 二、Net Proceeds（净收入）

在**已知售价**的模式下，Net Proceeds 的含义是卖家实际到手净收入：

```
Net Proceeds (USD) = 平台售价 - 平台佣金 - 物流费
```

**含义**：卖家从平台实际收到的钱（扣佣后）减去物流成本。
此值**不包含**采购成本和打包费，再减去这些和折损才是最终利润。

### 利润计算（已知售价模式）

```
利润 = Net Proceeds - 固定成本(USD) - 折损
     = 售价 - 佣金 - 运费 - 固定成本(USD) - 折损率 × 售价
```

### 计算示例

> 平台售价 $15，抽佣率 20%，国际运费 $18.5，采购+打包 ¥35 ($5.12)

| 项目 | 计算 | 金额 (USD) |
|------|------|-----------|
| 平台售价 | — | $15.00 |
| - 平台佣金 | $15 × 20% | -$3.00 |
| - 物流费 | 数据库查表 | -$18.50 |
| **= Net Proceeds** | | **-$6.50** |
| - 固定成本 | (33+2) / 6.84 | -$5.12 |
| - 折损 | $15 × 2% | -$0.30 |
| **= 利润** | | **-$11.92** |

> ⚠️ 当 Net Proceeds 为负数时，表示售价不足以覆盖佣金 + 运费，必然亏损。

---

## 三、反推功能

系统支持通过 Net Proceeds 反推平台售价：

```
平台售价 (USD) = (Net Proceeds + 运费USD) ÷ (1 - 佣金率)
```

在页面上输入期望的 Net Proceeds，系统会自动计算并填入对应的平台售价。
